## Instalation

### Dependencies 
**Arch**:
```
sudo pacman -S playerctl
```
**Debian**:
```
sudo apt install playerctl
```
### Installation 
```
git clone https://github.com/tasory/catfetch-ts/
cd catfetch-ts
chmod +x fetch.sh
./fetch.sh
```
Enjoy!
